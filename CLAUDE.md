# math-feedback-app

数学老师专属教学反馈工具，面向深圳校外培训机构的一对一及班课家长反馈场景。

## 技术栈
- 单文件 HTML（index.html），内联 CSS + JS
- 无需构建、无需 npm，双击即可打开
- 数据存储在浏览器 localStorage（key: mathFeedbackTool-v3）
- 适配 iPad Safari（主要使用设备）
- PWA 支持：manifest.json + sw.js 离线缓存 + 桌面快捷图标

## 双模式架构
- **经典模式**：左列表右编辑，全功能（CRUD / 生成 / 保存 / 同步 / 导出）
- **沉浸工作模式**：纯滑动手势操作，隐藏侧栏，卡片式逐个评价
- 双模式数据实时互通，任意切换不丢失进度

## 使用方式
- 双击 `index.html` 或 `node server.js` 启动本地服务器（http://localhost:8765）
- iPad Safari 打开后可「添加到主屏幕」全屏运行，像原生 App
- 主题切换：顶栏 🌓 按钮，支持浅色/暗色模式

## 沉浸工作模式手势
- 左滑 → 下一学生
- 右滑 → 上一学生
- 上滑 → 跳过当前学生
- 保存按钮 → 保存并自动跳转
- 全班完成 → 自动统计并退回经典模式

## 功能特性
- AI 反馈生成（本地 + 豆包润色），支持三部分结构化输出
- 档位预设 + 同档/全班同步
- 一键清理超过14天的旧记录（学生资料永久保留）
- 阶段性复盘提醒（累计4条记录触发）
- 历史记录管理：查看全部 / 逐条删除 / 清空
- 导出/导入 JSON 备份
- 企业微信一键复制跳转
- 评价小能手连续评价提示

## 注意事项
- 数据仅保存在浏览器 localStorage，定期使用「导出备份」按钮
- 豆包按钮会复制指令后跳转 App
- 项目结构已极简化，勿添加框架依赖
- 备份文件位于项目目录 index_backup_*.html
