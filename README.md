# 数学老师专属教学反馈工具

轻联网网页版反馈工具，适合深圳地区校外培训机构数学老师的一对一家长反馈场景。

## 主要功能

- 左侧固定学生目录，支持学段自定义标签
- 右侧学生页面编辑：姓名、家长称呼、薄弱知识点、课堂表现、优化档位、后续安排、自定义关键词
- 班级共享本周授课内容，切换学生不清空
- 生成纯文本反馈，复制后微信粘贴无格式
- 每位学生保留最近 5 条历史记录
- 鼠标和触屏支持同班左右翻页
- 快捷键：空格生成、Ctrl+C 复制、←/→ 翻页

## 使用说明

1. 将项目文件放在本地静态目录中（例如桌面目录）。
2. 打开 `index.html` 即可使用。
3. 左侧点击学生进入编辑。填写学生信息与班级授课内容。
4. 选择后续学习安排，点击「生成反馈」。
5. 点击「复制文案」即可复制纯文本反馈。

## 目录结构

- `index.html` - 页面入口
- `style.css` - 样式
- `app.js` - 业务逻辑与生成规则
- `README.md` - 项目说明

## 数据存储

本工具使用浏览器 `localStorage` 保存学生数据、班级授课内容、历史反馈，数据仅保留在本地。

## 注意

- 本版本为轻量网页版，适合本地部署或直接打开浏览器使用。
- 不含广告、不含浮夸效果、不含教师落款。
