# Math Feedback App Skill

## 项目身份
这是一个**深圳校外培训机构数学老师**的家长反馈生成工具。老师每周六/日给学生上 2 小时课，课后需微信发送教学反馈给家长。

## 技术铁律（严禁违反）
- **单文件 HTML**：index.html 内联 CSS + JS，不加任何框架、不引入 npm、不引入外部依赖
- **双击即用**：无构建步骤，iPad Safari 直接打开
- **数据存储**：仅浏览器 localStorage，key = `mathFeedbackTool-v3`
- **无后端**：不调服务器，无数据库，纯前端

## 班课 vs 一对一（必须严格区分）
- 一对一：绝对禁止「同学们」「大家」「班上的同学」等群体词，反馈按专属单独辅导视角
- 班课：不刻意营造私教感，但保留课堂对个体的针对性观察
- 侧边栏二级标题用彩色圆点区分：🔵蓝色 = 班课，🟢绿色 = 一对一

## 学段规则
| 学段 | 教材 | 特殊限制 |
|------|------|---------|
| 小学1-5年级 | 北师大版 | 禁止：自律、自主复习规划、自我管理；不涉及余角补角 |
| 六年级 | 北师大版 | 偶尔提小升初衔接即可，不反复刷必考模块；不涉及余角补角、折叠几何角度变化 |
| 初中 | 北师大版数学 / 深圳人教版化学 | 无额外限制 |

## 场景限定
- **仅针对周六/周日 2 小时课程**，不涉及学生在校日常、平日行为习惯
- **禁止描写平日日常**：所有评语只看周末课堂表现

## 硬性禁用词汇（所有 AI 润色 prompt 必须包含）
```
奠基、夯实根基、构筑、赋能、底层逻辑、逻辑链条、知识架构、多维分析
聚焦、自律、自主规划、平日、日常、每天、每日
今天我们来讲、所以咱们、家长您、我我我、带他
```
- **自主复习**：小学生禁止，初中生不禁止
- **天道酬勤、上善若水**：老年风格文学句禁止

## 易错点铁则（所有 AI prompt 必须包含）
```
严禁编造学生犯简单基础错误（记错平角/周角度数、分不清射线直线、
忘记三角形内角和、弄混平行四边形梯形等）。
这些极低幼错误正常学生不会犯，写了显得老师在乱说。
只有用户在"需要注意改进的地方"自定义框里明确输入了具体问题才可提及，
且必须用"容易忽略""需要更仔细""偶尔看漏"代替"记错""分不清""不会"。
```

## 反馈格式（所有 AI prompt 必须锁定）
```
开头固定：xx爸爸/妈妈/家长您好，下面是xx本周的课堂反馈：
一、本周学习内容：（原样不动，禁止修改）
二、课堂表现：（口语化+1句文雅不老气文学句+不排列标签）
三、后续学习安排：（具体可落地+不命令家长+选了作业标签才提作业）
```

## 禁止的反馈内容
- 错题汇总、每日固定X分钟复盘、每天打卡类建议
- 要求家长做的事情
- 教材版本、考纲相关词汇
- 「必考」「高频考点」反复堆砌

## 文学句规则
- 45+ 句池，每班独立轮换，不重复
- 必须自然嵌入上下文，不单独插句
- 风格：稳重文雅，不鸡汤不幼稚不浮夸

## 标签池规则
- 课堂表现、表现优点、需要改进 各有约25个标签
- 禁止描写学生日常行为习惯（坐姿、平日习惯等）
- 禁止出现「自律」相关的标签（适用各学段）
- 禁止「课后需要」「习惯有待培养」等平日相关表述
- 小学生标签池额外禁止自主复习等词汇

## 数据结构
```
groups → subGroups → students（三层树）
students: name, gender, parent(爸爸/妈妈/家长), grade, sem, tier
tier: 优秀档/中等档/进步档（三档预设自动同步）
history: 每生最多4条
lessonContents: 每二级标题共享
```

## 交互行为
- 切学生：自动替换预览中的人名/称呼/性别代词
- 同档同步：同步同班同档位学生的全部预设（不含姓名性别家长称呼）
- 全班同步：同步授课内容
- iPad：+ 按钮常显；电脑：hover 显示
- 一键清空：只清本周数据（授课内容+勾选+预览），档案和历史保留
- 保存：必须点保存才写历史；生成 ≠ 保存

## UI 风格
- 莫兰迪色系：--bg #ece6e1, --accent #8f7a6f 等
- 9 只小黄兔 SVG 装饰，低透明度
- 豆包按钮蓝白配色，图标猫头鹰
- 序号圆形 section-num，连续不跳号
- 圆角卡片 + 柔阴影
